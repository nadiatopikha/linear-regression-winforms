# Linear Regression WinForms App

Desktop application written in C# using WinForms for data analysis and linear regression modeling.

## Features

- Import numeric data from TXT file
- Generate synthetic datasets
- Calculate correlation and covariance
- Estimate linear regression parameters using gradient descent
- Visualize data and regression line using charts

## Technologies

- C#
- .NET
- Windows Forms
- DataVisualization Chart

## How to use

1. Load a TXT file with numeric data
2. Select vectors for analysis
3. Run regression calculation
4. View results on the chart
